# versus

제작: 20210320 안선우

기컴프 개인과제로 만들어진 게임입니다.

!!표시가 나왔을 때 재빨리 일정 시간 내에 화면 아무 곳을 클릭하면 상대를 이길 수 있고,

표시가 나오기 전이나 너무 늦게 누르면 패배하는 반응속도 게임입니다.
